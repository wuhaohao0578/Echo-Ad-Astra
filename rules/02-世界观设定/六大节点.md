# 六大坐标节点

**THE SIX KNOWN NODES**

截至2143年，SIGMA已公开6组坐标节点。每个节点都带来了新的发现——但也带来了新的问题。以下为公开信息概要。

> **注意**：六大节点的具体设定（行星环境、生态系统、特有资源、异常现象等）将在星际穿越指南（Chapter 5）中详细展开。此处仅列出框架性信息，供世界观参考。

## Node I - Alpha

**公开年份：** 2063/2090
**距离：** 4.6 光年
**状态：** 完全开发

半人马座α星系。人类抵达的第一颗异星。第一批谐素矿脉的发现地。经过数十年的开发，Node Alpha已从荒芜的矿源行星发展为人类最大的星际前哨基地，拥有永久性定居设施和完整的工业链。第一章马丁执行任务的"Alpha前哨站"即位于此。

## Node II - Borealis

**公开年份：** 2128
**距离：** 公开信息
**状态：** 大规模开采中

第二组公开坐标。更大规模的谐素矿脉。首次发现非地球来源的微生物级生命形式——一种以谐素矿脉中的微量辐射为能量来源的极端嗜热古菌类似物。这一发现极大地改变了天体生物学的范式。

## Node III - Caldera

**公开年份：** 2131
**距离：** 公开信息
**状态：** 有限开采

行星环境极端恶劣。发现了一种新型谐素变体——结晶结构与标准谐素不同，退相干抑制特性更强但更不稳定。同步员在此执行任务时，同步率波动幅度异常增大。原因不明。

## Node IV - Drift

**公开年份：** 2134
**距离：** 公开信息
**状态：** 探索阶段

首次在坐标行星上发现宏观复杂生命体——一类具有群体智能行为的外骨骼节肢动物。它们没有可辨识的感觉器官，但能精确定位同步员的素体。SIGMA将其归类为"本能应激反应"，但部分研究人员对此存疑。

## Node V - Echo

**公开年份：** 2138
**距离：** 公开信息
**状态：** 受限探索

与游戏同名的节点。行星表面存在大面积的、疑似非自然形成的对称地质结构——太规律了，不像侵蚀或板块运动的产物。更像是……遗迹。但不是人类的遗迹。SIGMA对Node Echo实施了比其他节点更严格的访问管制。官方理由是"环境危险等级较高"。

## Node VI - Forge

**公开年份：** 2141
**距离：** 公开信息
**状态：** 初步勘探

最近公开的坐标。初步勘探报告中有大量内容被SIGMA以"安全审查"为由删除。公开信息仅显示：该行星具有异常强烈的电磁背景辐射，对素体的纳米结构产生了前所未有的干扰模式。首批勘探队报告了"通讯设备自发接收到结构化数据"的现象——但相关数据在回传地球前被SIGMA加密封存。
