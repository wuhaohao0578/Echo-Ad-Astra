# Echo Ad Astra 世界观设定

**修订版 3.8 · 文档等级：SIGMA 内部参考资料 · 部分脱敏**

## 概述

公元2058年，一位年仅20岁的射电天文学研究员陈（Chen）在例行巡天观测中，标记了一组无法归类的异常信号。八十五年后的今天，人类依靠这组信号的遗产，将意识投射到了数十光年之外的异星前线。

这份文档记录了从信号发现到当前游戏纪元（公元2143年）的完整历史脉络。

## 关键概念速查

### 谐素（Lucidine）
一种仅存在于信号坐标指向行星上的超稳态结晶矿物，能够在宏观尺度上抑制量子退相干，是跨星际意识投射技术的唯一可用介质。

### SIGMA
独立超主权机构，拥有坐标数据的唯一管辖权和谐素提炼核心技术。

### 同步员
具有"向性和谐"生理兼容性的个体，能够通过谐素介质将意识投射到异星素体中。

### 坐标危机
2121-2123年，各方势力为争夺谐素资源几乎引发全面战争，最终由陈以自杀威胁迫使各方接受SIGMA独立。

### 广播指数
每次使用谐素都会产生量子信号涟漪，这些信号与原始的SIG-0058信号处于同一频域——使用谐素就是在向宇宙深处"回话"。

## 时间线要点

- **2058** - 陈发现SIG-0058信号
- **2061** - 陈破解坐标，发现数百组星系坐标
- **2063** - 远征启航前往Node Alpha
- **2090** - 发现谐素矿脉
- **2119** - 发现退相干抑制特性
- **2121-2123** - 坐标危机
- **2124** - 《星际坐标管理公约》签署，SIGMA改组
- **2125** - 首次成功的意识纠缠投射
- **2143** - 当前游戏纪元

## 六大节点

1. **Node Alpha** - 半人马座α，人类第一个星际前哨基地
2. **Node Borealis** - 发现外星微生物
3. **Node Caldera** - 发现谐素变体，同步率异常
4. **Node Drift** - 发现宏观复杂生命体
5. **Node Echo** - 疑似非自然地质结构，受限探索
6. **Node Forge** - 异常电磁辐射，部分信息被封存
