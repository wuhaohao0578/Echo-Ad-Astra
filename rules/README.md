## 🎲 核心机制速览

### 天平系统（12D6 骰池）
- 玩家和 GM 共享 12 颗骰子
- 玩家骰数 = 6 + 行动等级 + 判定加值 - 阻力
- 5-6 为成功，比较双方成功数

### 同步率机制
- 安全区：80%-120%
- 低于 80%：素体性能下降
- 高于 120%：意识过载，增加广播指数

### 双轨进度尺
- 蓝色进度轨：任务进展
- 红色威胁轨：危机逼近
- 两者相遇时触发转折事件

### 心能点数
- 最重要的资源
- 用于突破困境、重投骰子
- 上限由三重锚点决定
