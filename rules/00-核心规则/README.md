## 核心机制

- **天平系统**：12D6 骰池零和博弈判定系统
- **双轨进度尺**：蓝色进度轨 vs 红色威胁轨
- **同步率**：80%-120% 安全区，超出会有后果
- **心能点数**：最重要的资源，用于突破困境
