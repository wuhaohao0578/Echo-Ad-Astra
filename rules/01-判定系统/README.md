# 判定系统

天平系统（Libra System）- 零和博弈判定机制

## 核心理念

在《星海回响》中，所有的冲突与挑战不再是玩家单方面的测试，而是与对象的零和博弈。主持人投掷的骰子不单纯指代 NPC 的技巧或者物品的难易程度，而是代表命运对玩家的阻碍。

## 快速参考

### 基础判定流程

1. **宣告意图**：你想做什么？想达成什么效果？
2. **评估难度**：GM 根据环境与威胁决定阻力难度
3. **分配骰池**：
   - 玩家骰数 = 6 + 行动等级 + 判定加值 - 阻力
   - GM 骰数 = 12 - 玩家骰数
4. **投掷骰子**：5-6 为成功
5. **结算**：比较双方成功数

### 判定结果

- **玩家成功数 > GM 成功数**：判定成功，完美达成目标
- **玩家成功数 = GM 成功数**：部分成功，达成目标但付出代价
- **玩家成功数 < GM 成功数**：判定失败，未实现目标

### 阻力难度速查

| 难度 | 阻力值 | 关键词 | 示例 |
|------|--------|--------|------|
| 如鱼得水 | +2 | 敌人犯蠢，环境完美 | 敌人昏迷，在你开发的系统中操作 |
| 优势 | +1 | 有利条件 | 高处视野，拥有完整蓝图 |
| 常规 | 0 | 标准作业 | 打靶练习，撬开普通挂锁 |
| 麻烦 | -1 | 轻度打扰 | 对抗普通敌人，暴雨中驾驶 |
| 困难 | -2 | 标准战斗难度 | 标准遭遇战，昏暗灯光下射击 |
| 严峻 | -3 | 精英级难度 | 对抗精英单位，拆除倒计时炸弹 |
| 极难 | -4 | Boss 级难度 | 对抗传奇佣兵，入侵核心数据库 |
| 绝境 | -5 | 战役高潮难度 | 对抗类龙生物，手写代码重置超级 AI |
| 传奇 | -6 | 物理与运气的极限 | 肉身拆高达，狙击几公里外高速导弹 |

## 机制原则

《星海回响》的所有机制都是为了将游戏进行下去。玩家运用机制，与主持人一起将独属于你们的故事演绎完成才是这本规则的初心。

**如果任何机制与游玩者的实际需求相悖，在参与者都同意的情况下，去修改任何机制和规则都是可行的。**

但在确定机制修改前，请一定听取和尊重所有参与者的意见。
